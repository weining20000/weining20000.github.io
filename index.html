Project Experience

Machine Learning

2016 New York City Poverty Studies
Role: EDA, ggplot2 visualization, Logistic Regression implementation, and results assessments
This project built Generalized Linear Model (GLM) and Random Forest using R as the programming language to predict poverty rates in New York City. The findings show that Income, Unit Population, Family Type, Housing Type and Borough have the strongest predictive power for poverty status. Random Forest appears to be a better model in this instance with an area under the curve (AUC) score of 0.90 and accuracy score of 84%. [Presentation] [Code]

2017 Barcelona Traffic Accident Severity Prediction
Role: data cleaning, EDA, and Random Forest implementation
This project implemented Decision Tree and Random Forest to explore the determinants of traffic accidents in Barcelona. Particularly, it examined how location, transportation type, time, and unemployment rate are correlated with traffic accidents. Both the Random Forest model and the Decision Tree model yielded a 90% accuracy. [Code]

Flooding Damage Detection from Post-Harvey Hurricane Satellite Imagery based on CNN
Role: project lead (pre-trained models implementation, performance matrices construction, and report writing)
This project desiged and trained a convolutional neural network from scratch and compared it with two existing neural networks, the VGG-16 and Resnet50, for a binary classification problem (to differentiate images that contain flooding damages from the ones still intact.) The best model (a custom CNN with fewer convolutional layers) was able to achieve an accuracy score of over 90% on the hold-out test set. The networks were trained using PyTorch as the framework with a single NVIDIA K80 Tesla GPU via the Google Cloud Platform. [Report] [Code]
