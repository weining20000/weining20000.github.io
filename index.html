Machine Learning

Flooding Damage Detection from Post-Hurricane Satellite Imagery based on CNN
Authors: Bixuan Huang and Weining Hu
In this project, we design and train a convolutional neural network from scratch and compare it with two existing neural networks, the VGG-16 and Resnet50, for a binary classification problem (to differentiate images that contain damages (i.e. flooding area or damaged buildings) from the ones still intact.) Our best model (a custom CNN with fewer layers) is able to achieve an accuracy of over 90% on the hold-out test set. We train our networks using the PyTorch framework with a single NVIDIA K80 Tesla GPU via Google Cloud Platform (GCP). [Report] [Code]

Malaria Infected Cell Classification via MLP
Author: Weining Hu
This project used Keras to build an MLP model to classify the cells into 4 different categories, which are: “red blood cell”, “ring”, “schizont”, and “trophozoite.” The best model over the seven-day data challenge competition (mid-term exam for Machine Learning II) has achieved a reasonable performance with a 0.6147 macro-averaged F1-score and 0.6212 Cohen’s Kappa Score. The average score of the two is 0.6179. [Report] [Code]

Multi-Label Classification of Blood Cells using CNN
Author: Weining Hu
This project used PyTorch to build a CNN model to recognize all types of cells that are present in the given images. These cell types are: red blood cell, difficult, gametocyte, trophozoite, ring, schizont, and leukocyte. The best model over the seven-day data challenge competition (final exam for Machine Learning II) yielded a loss score of 4.54085. [Report] [Code]

2017 Barcelona Traffic Accident Severity Prediction
Authors: Nhung Nguyen and Weining Hu
This project implemented decision tree, random forest, and k-nearest neighbors (KNN) to explore the determinants of traffic accidents in Barcelona. Particularly, we examined how location, transportation type, time, and unemployment rate are correlated with traffic accidents. Both of the random forest model and the decision tree model yielded a 90% accuracy, achieving better results than KNN. [Code]

2016 NYC Poverty Studies
Authors: Isaac Amoah, Evan Carraway, Victor Chen, Weining Hu, and Tina Nguyen
This project built General Linear Regression model and Random Forest using R as the programming language to predict poverty rate in New York City. The study of poverty is important because it can help decision makers better understand this ongoing problem to break the cycles of poverty and prioritize effective interventions for the most at-risk groups. Our findings show that Income, Unit Population, Family Type, Housing Type and Borough have the strongest predictive power for poverty status. Random forest appears to be a better model in this instance with an AUC score of 0.9 and accuracy score of 84%. [Presentation] [Code]
